pour vous connecter sur le site et mettre un tuto en ligne : 

- cliquez sur create an account et rentrez ce que vous voulez en pseudo mais un mot de passe d'au moins 6 caracteres.

pour mettre un tuto en ligne avec une photo :

- rentrez une url d'image en ligne (google images...) et le reste dans les input correspondants.

la base de donnée se met à jour et les tutos sont visibles.
